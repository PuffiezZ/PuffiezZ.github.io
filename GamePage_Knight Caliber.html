Knight 5.56 Calibur is a 2D top-down, roguelike shooter with pixel-style graphics. This was my first game development project, and it played a major role in inspiring my passion for creating games.
The First game I've made
I learned a lot about how to make games from this project. I created it using Unity, which required me to learn the basics of C# and Unity’s tools. I also learned how to work with sprite sheets and implement them in the game. I released a demo as well — you can play it on my personal itch.io account!
